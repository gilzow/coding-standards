# Frontend guidelines

## File structure

Structure your files to help keep things tidy for the backend people.

	SITE ROOT
		
		- styles
			- less/sass (source)
			- css (compiled/minified code)
				- screen.css
			- fonts 
		
		- scripts
			- js (source)
			- min (compiled/minified code)
				- script.js
		
		- images
			- template (images needed for the site template)
